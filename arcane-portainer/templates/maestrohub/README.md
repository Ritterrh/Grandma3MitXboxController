# MaestroHub

Industrial data orchestrator with a Unified Namespace, drag-and-drop pipelines, and 40+ protocol connectors. Built-in MCP server brings AI agents into your operational data stack.

## Notes

Welcome to MaestroHub!

• Open http://&lt;host&gt;:8080 to create your admin account
• MQTT clients can connect to &lt;host&gt;:1883 (TCP) or &lt;host&gt;:8083 (WebSocket)
• Data persists in the maestrohub-data Docker volume — back it up before destroying the stack
• Get started guide: https://docs.maestrohub.com/overview/get-started
